## 详细版（App Store / 产品页面）

### 中文版

**Claude Hub - Claude Code 会话管理器**

还在终端里苦苦寻找之前的 Claude Code 对话？Claude Hub 是专为 Claude Code 用户打造的会话管理工具，让你随时随地浏览、搜索和管理所有编程对话。

**✨ 核心功能**

🔍 **全文搜索** - 快速找到任何对话中的代码片段和讨论  
🏷️ **智能标签** - 给会话添加标签，轻松分类管理  
⭐ **收藏归档** - 重要会话一键收藏，完成项目归档整理  
📝 **自定义命名** - 给会话起个好记的名字  
📊 **统计仪表盘** - 可视化你的编程活动  
📅 **日历视图** - 按日期浏览会话历史  

**🖥️ 多端支持**

- **macOS 桌面端** - 原生 Electron 应用，流畅体验
- **iOS 移动端** - 随时随地查看你的编程对话
- **远程服务器** - 通过 SSH 管理远程机器上的 sessions

**💡 特色亮点**

- 📴 **离线模式** - 同步后即使没有网络也能查看历史会话
- 🔄 **增量同步** - 只同步新更新的会话，快速高效
- 🌙 **深色模式** - 为程序员打造的护眼界面
- 🔐 **本地优先** - 你的数据只存在本地，隐私安全

### English Version

**Claude Hub - Claude Code Session Manager**

Still digging through terminal history to find that Claude Code conversation? Claude Hub is the ultimate session manager for Claude Code users, letting you browse, search, and manage all your coding conversations anytime, anywhere.

**✨ Core Features**

🔍 **Full-text Search** - Quickly find any code snippet or discussion

📊 **Stats Dashboard** - Visualize your coding activity

- **macOS Desktop** - Native Electron app, smooth experience
- **iOS Mobile** - View your coding sessions on the go
- **Remote Servers** - Manage sessions on remote machines via SSH

**💡 Highlights**

- 📴 **Offline Mode** - View synced sessions without network
- 🔄 **Incremental Sync** - Only sync new updates, fast and efficient
- 🌙 **Dark Mode** - Eye-friendly interface for developers
- 🔐 **Local-first** - Your data stays on your device, privacy guaranteed

## 关键卖点总结

1. **痛点明确** - 解决在终端翻找历史对话的问题
2. **移动端支持** - 手机上查看编程对话是刚需
3. **离线可用** - 外出时也能查看已同步的数据
4. **开发者友好** - 深色模式、标签分类、全文搜索
5. **隐私安全** - 本地存储，数据不上云
